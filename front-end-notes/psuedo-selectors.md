#Psuedo-class selectors

###A CSS pseudo-class is a keyword added to selectors that specifices a speical state of the element to be selected.

##### Example
```css
 selector: pseduo-class {
  property: value;
  };
```
Example 2
```css
img: hover {
 color: #000000;
}
```
